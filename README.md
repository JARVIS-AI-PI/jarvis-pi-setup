# Jarvis AI Setup

## How to Install

1. Clone this repo to your Raspberry Pi Desktop:
   ```
   git clone https://github.com/JARVIS-AI-PI/jarvis-pi-setup.git ~/Desktop/Jarvis
   ```
2. Run the installer:
   ```
   ~/Desktop/Jarvis/install.sh
   ```
3. Reboot:
   ```
   sudo reboot
   ```
4. Click the Jarvis AI icon on your Desktop to launch.
